# Retro Strategy Game Simulator

A 90s-style tactical command center game built with HTML, CSS, and JavaScript. Inspired by classic Star Trek games but with modern touches for a 10-year-old audience.

## Features
- Grid-based map system (quadrants)
- Enemy detection and tracking
- Missile/battalion command mechanics  
- Visual feedback with animations
- Retro Windows 95/98 aesthetic

## Technical Details
- Single HTML file (self-contained)
- Canvas for tactical map display
- CSS for retro UI styling
- Vanilla JavaScript for game logic

## Target Audience
Designed for a 10-year-old with intuitive mechanics and engaging visual feedback.

## How to Play
1. Open `index.html` in any modern web browser
2. Navigate the grid using arrow keys or click-to-move
3. Scan quadrants for enemies using sonar
4. Launch missiles at detected enemies
5. Destroy all enemy units to win
